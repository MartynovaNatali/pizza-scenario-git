# Сценарий использования: Заказ пиццы с топпингом домой, онлайн-оплата

## Актор
Зарегистрированный пользователь

## Цель
Получить горячую пиццу с дополнительным беконом по домашнему адресу

## Основной поток

1. Вход в систему
2. Выбор пиццы «Пепперони»
3. Добавление топпинга «бекон»
4. Переход в корзину и проверка суммы
5. Выбор адреса «домой»  
6. Выбор оплаты картой онлайн
7. Нажатие «Оплатить и заказать»
8. Успешная оплата  

## Альтернативные сценарии

### Альтернатива 1: Оплата наличными курьеру
- Заказ фиксируется без онлайн-транзакции
- Курьер принимает оплату при доставке

### Альтернатива 2: Доставка в офис
- Вместо домашнего адреса выбирается офис
- Реквизиты офиса подставляются из профиля

## Исключения

### Исключение 1: Недостаточно средств на карте
- Платёжный шлюз возвращает ошибку
- Пользователь выбирает другой способ оплаты

### Исключение 2: Нет свободных курьеров
- Система показывает сообщение: «Доставка недоступна сейчас, попробуйте позже»
- Заказ не создаётся
